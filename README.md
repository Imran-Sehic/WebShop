# WebShop
<h3>This is a WebShop application built in JSF-Java Server Faces framework.</h3></br></br>
The application contains a login and a register page. Besides that a user can buy products other users inserted, delete or buy more products of same kind in the basket page, delete it's own products from the web-shop and update or delete it's own user profile.</br></br>
The application uses Hibernate and JPA to communicate to the mySQL database where all the data is stored.</br></br>
